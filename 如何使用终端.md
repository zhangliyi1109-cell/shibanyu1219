# 💻 如何使用终端

## 📖 什么是终端？

**终端（Terminal）** 是一个**命令行界面**，你可以通过输入文字命令来控制计算机。

想象一下：
- **图形界面** = 用鼠标点击图标
- **终端** = 用键盘输入命令

---

## 🖥️ 如何在 Windows 上打开终端？

### 方法 1：在 Cursor/VS Code 中打开（最简单）⭐

1. **按快捷键**：`Ctrl + ~`（Ctrl 键 + 波浪号键）
2. **或者**：点击底部工具栏的 **"Terminal"** 标签
3. **或者**：菜单栏 → View → Terminal

### 方法 2：Windows PowerShell

1. 按 `Win` 键（Windows 键）
2. 输入 `powershell`
3. 点击 "Windows PowerShell"

### 方法 3：命令提示符（CMD）

1. 按 `Win + R`
2. 输入 `cmd`
3. 按回车

### 方法 4：在文件夹中打开

1. 打开文件资源管理器
2. 进入你的项目文件夹
3. 在地址栏输入 `powershell` 并回车

---

## 👀 终端长什么样？

终端通常显示：

```
PS D:\ALL WORKS\AIbuild\program\shibanyu\shibanyu1219> _
```

- `PS` = PowerShell 提示符
- `D:\...` = 当前文件夹路径
- `>` = 命令提示符
- `_` = 光标（闪烁的）

---

## ⌨️ 如何使用终端？

### 基本操作：

1. **输入命令**：在光标处输入文字
2. **执行命令**：按 `Enter`（回车）键
3. **查看结果**：命令执行后会显示结果

### 示例：

```bash
# 查看当前文件夹内容
dir

# 查看 Git 版本
git --version

# 查看当前路径
pwd
```

## 🎯 现在你需要做的：

### 1. 打开终端

在 Cursor 中按 `Ctrl + ~`，或点击底部的 "Terminal" 标签

### 2. 确认你在项目文件夹

终端应该显示类似：
```
PS D:\ALL WORKS\AIbuild\program\shibanyu\shibanyu1219>
```

如果不在，执行：
```bash
cd "d:\ALL WORKS\AIbuild\program\shibanyu\shibanyu1219"
```

### 3. 执行命令

现在你可以执行 Git 命令了！

---

## 📝 接下来要执行的命令：

### 步骤 1：在 GitHub 创建仓库

1. 打开浏览器
2. 访问：https://github.com/new
3. 创建仓库（参考之前的说明）

### 步骤 2：在终端执行（创建仓库后）

```bash
git remote add origin https://github.com/your-username/shibanyu.git
git branch -M main
git push -u origin main
```

---

## 💡 提示

- 终端中的命令是**区分大小写**的
- 输入命令后必须按 `Enter` 才会执行
- 如果命令执行失败，会显示错误信息
- 可以复制粘贴命令到终端
